The "Fizz-Buzz" Test

Write a program that prints the numbers from 1 to 100. But for multiples of three print “Fizz” instead of the number and for the multiples of five print “Buzz”. For numbers which are multiples of both three and five print “FizzBuzz”.

A SIMPLE QUESTION

Write a function that gets an array of strings and returns the string with the most occurrences.

Understand the question

Repeat the question

“So you want me to find the string that appears the most in the array?”

Ask questions

“May I use temporary variables?”
Interviewers sometimes tell you the question wrong and don't admit it - this helps them

Get comfortable

thinking out loud

We want to see your thought process

Start with the most naive and simple solution and make sure to communicate that

Progressive Solving

Start with the most naive and simple solution, communicate it and move to a better one

Show dry runs of your code

Draw arrays, show progress, debug it on paper

// Sum an array

x sum x + sum

Clean Code

Re-write your solution, do not submit your drafts

Clean code: clear text, good indentation, good variable names

function reverse(string) {
  return string
    .split('')
    .reverse()
    .join('');
}

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
    console.log( '"customevent" has fired' );
} );
